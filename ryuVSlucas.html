Ryu VS Lucas

- Mini game using ESP32 -

For this project, we were tasked with creating a mini-game using two custom made ESP32 boards. With my friend Lucas we decided to develop a 1-vs-1 fighting game. We started by designing two pixel art characters, each equipped with four different moves, laying the foundation for an engaging and competitive gameplay experience.

We first tried to display the characters on the 128x64 screen and make them move when pressing a button. Initially, we wanted to control the characters' movements with a joystick, but we couldn't display it correctly. So, we decided to focus solely on the attacking mechanics.

We used WebSocket to connect both boards by creating a server on Lucas's board and connecting my board to it.
We successfully created a mini-game with the two boards, where the objective was to press the button as fast as possible to drain the opponent's life point and secure victory. However, we couldn't implement the character animations in the game, which was a bit disappointing.
Overall, even though it was a bit challenging, it was a really fun project to work on.
