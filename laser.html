Laser Shapes

- Like a Fireflies Ballet -

For this project, my goal was to generate an output from an input using Arduino. I chose to use two servo motors as the output, controlled by a joystick, with a laser attached to the servos. By capturing long-exposure photos, I wanted to create light drawings in the air with the laser.

I first researched how to connect the servo motors and joystick to the Arduino board. Then I had to figure out how to make the motors move along the X and Y axes, determining the best way to attach one servo to the other and mount the laser for smooth and precise movement.
The horizontal left-right movement of the joystick controlled a servo motor used as a base, while the vertical up-down movement controlled a second servo motor mounted on top of the first. This setup allowed for two-axis control of the laser.

I first experimented by taking pictures using only the laser, moving it by hand. To create a dark environment, I went to the bathroom, where the laser reflections on the tiles produced fascinating images. Controlling the motors with the joystick was a bit challenging, but despite that, the setup still generated interesting light patterns. The servo motors were not very precise, and their movements were quite abrupt, which mainly resulted in grid-shaped patterns.

Lastly, I made an automated code to create a flower-shaped movement, allowing the servos to draw a flower with the laser. However, since the servos were not stable, each repeated movement resulted in random zigzag lines, adding an unexpected dynamic effect to the pattern.
